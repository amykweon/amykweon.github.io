    2022

    Mutimodal Research on Radicalization

    Research Assistant, Spoken Language Lab, Columbia, New York

    Analyzing videos about domestic radical organizations that are uploaded in media platform such as Youtube or Bitchute. The targetted radical groups are QAnon, Proud boys, Oath Keepers, and BLM. The videos include various genre from news report, documents to vlogs made by radical group members. The videos were analyzed to identify multimodal features that make the video persuasive.

    • Extracted facial emotion features and studied the patterns in radicalizing and deradicaling content
    • Extracted transcripts and ran textual sentiment analysis
    • Extracted metadata of video (likes and views) and measured popularity
    • Studied the relationship between the popularity measure and the multimodal features
    • Studied the correlation between the pattern in metadata (increase in video uploads or comments) and the real-life event

    Publication: Unveiling the Influencers of Radical Content: A Multimodal Analysis of QAnon Videos, IC2S2 2023. [pdf]

    Dabor

    Programming Language Translator

    Design a new language and create an appropriate compiler with LLVM and OCaml. The proposed language, Dabor, aims to offer programmers an intuitive and efficient means of creating board games. This language will provide flexibility to create not only a conventional board game, such as Checkers but also user-customized games with new rules.

    • Designed and build scanner and parser for the language such that the code is tokenized and parsed.
    • Wrote AST, SAST and code generation algorithm that implements the accurate execution of the language.
    • Created a test set that can test all implementation and error outputs.
